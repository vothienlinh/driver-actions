# driver-actions
Videos containing driver distraction activities
## Dataset
Due to the large file size of the high-resolution videos, the dataset is hosted on Google Drive.
* **Download Link:** [Download Dataset Here](https://drive.google.com/drive/folders/19FjfSf-UCGMALLfrA-kPYbpg723wY4O2?usp=drive_link)
* **Total Size:** ~4.3 GB
* **Organization:**
